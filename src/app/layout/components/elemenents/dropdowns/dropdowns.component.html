Directions

You can use dropdown-menu from diffrent direction to use dropdown-menu from bottom right use .dropdown-menu-right along with ngbDropdownMenu directive. For dropup use [placement]="'top'" directive along with ngbDropdownToggle directive. For dropright use .dropright & [placement]="'right-top'" directive along with ngbDropdownMenu directive. For dropleft use .dropleft and [placement]="'left-top'" along with ngbDropdownMenu directive.

To create a dropdown-menu with groups you can use .dropdown-header for the header of groups and for sepration of group you can use .dropdown-divider.
